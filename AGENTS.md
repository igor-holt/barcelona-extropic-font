# B-E Deploy — Agent Handoff
> For: claude-cli / gemini-cli / codex-cli
> All steps are idempotent. Resume from any failure point.

## State

Completed steps are tracked in `.deploy_state`.  
Delete it to restart from scratch. Delete a line to re-run that step.

## Quick Deploy

```bash
# Set your gateway source directory and URL, then run:
export GATEWAY_DIR=/path/to/coalition-gateway
export GATEWAY_URL=https://coalition-gateway.YOUR_SUBDOMAIN.workers.dev

./scripts/deploy.sh
```

## Step-by-Step (if deploy.sh fails at any step)

```bash
# 1. Prerequisites
npm i -g wrangler
wrangler login

# 2. WASM toolchain
rustup target add wasm32-unknown-unknown
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh

# 3. Tests — MUST PASS before deploy
cargo test --features ambigraph

# 4. WASM build
wasm-pack build --target web --release --features ambigraph --out-dir pkg

# 5. R2 bucket
wrangler r2 bucket create be-font-assets

# 6. Upload assets
wrangler r2 object put be-font-assets/v1.0.0/barcelona-extropic.wasm \
  --file pkg/be_ambigraph_bg.wasm --content-type application/wasm \
  --cache-control "public, max-age=31536000, immutable"

wrangler r2 object put be-font-assets/v1.0.0/barcelona-extropic.js \
  --file pkg/be_ambigraph.js --content-type application/javascript \
  --cache-control "public, max-age=31536000, immutable"

wrangler r2 object put be-font-assets/v1.0.0/barcelona-extropic.css \
  --file gateway/barcelona-extropic.css --content-type text/css \
  --cache-control "public, max-age=31536000, immutable"

# 7. Patch coalition-gateway (two manual edits to src/index.ts)
#    Line 1 — top of file:
#      import { handleFontRoute } from './font_route';
#    Line 2 — inside fetch(), before the OPTIONS handler:
#      if (path.startsWith('/fonts/')) return handleFontRoute(request, env);
#    Then copy font_route.js:
cp gateway/font_route.js $GATEWAY_DIR/src/font_route.js

# 8. Deploy gateway
cd $GATEWAY_DIR && wrangler deploy

# 9. Smoke test
GATEWAY_URL=https://your-gateway.workers.dev ./scripts/smoke_test.sh
```
